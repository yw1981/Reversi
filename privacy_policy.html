Privacy Policy of TicTacToe

This application collects some personal data from its users. The user can optionally login to Facebook, in order to invite their Facebook friends for a match of TicTacToe. The application requests the Facebook permission to retrieve the user's friends. The list of Facebook friends is not stored. The application only stores the user's Facebook ID and display name.

We don't sell or rent Users' Data to any third parties for any purpose. The only uses of Data are those highlighted in this policy. Users are the only owner of their Data and can request modification or deletion at any time.
